# Security Policy

## Supported Versions

The table below outlines the versions of **VishwaGuru** that are currently receiving security updates.

| Version | Supported          | Notes |
| ------- | ------------------ | ----- |
| 1.0.x   | :white_check_mark: | Active development / Current stable release |
| < 1.0   | :x:                | Historical experimental branches |

## Reporting a Vulnerability

We take the security of VishwaGuru seriously. If you find a security vulnerability, please do not report it through public issues. Instead, please follow the steps below:

1. **Email the Maintainers**: Send a detailed report of the vulnerability to the project maintainers.
2. **Details to Include**:
   - Description of the vulnerability and its potential impact.
   - Step-by-step instructions (or a proof-of-concept script/exploit) to reproduce the issue.
   - Any proposed remediation or fixes.
3. **Response Time**: You will receive an initial acknowledgment of your report within **48 hours**. We will keep you updated as we investigate and work on a fix.
4. **Responsible Disclosure**: We kindly ask that you give us reasonable time to resolve the vulnerability before publishing details publicly.
