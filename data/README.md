# Data Directory

This directory contains static data files used by the VishwaGuru application.

## Files

### `responsibility_map.json`
This JSON file contains the mapping of civic issues to the responsible authorities and government ministries. It is used by the backend to identify which department should be notified for a specific type of issue (e.g., Potholes -> Municipal Corporation).

## Usage
These files are loaded by the backend application at runtime. Ensure this directory is included in your deployment artifacts if you are deploying the backend manually.
